# Tinder_clone
Création d'un clone tinder.

## Pour lancer l'application
`cd Front && npm i && npm start`  
`cd Backend && nodemon server.js ` 

## Technologies utilisées
* MongoDB
* Express
* React
* Node
* CSS3
* Material UI
* Firebase
* Axios
